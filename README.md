# App de Reparto Offline

Aplicación Android para gestión de repartos con escaneo QR offline.

## Características
- Escaneo de códigos QR offline
- Detección automática de zonas
- Gestión local de paquetes
- Estadísticas por zona
- Carga manual de datos
- Interfaz profesional

## Requisitos
- Android 8.0 (API 24) o superior
- Cámara para escanear QR
- 50MB de espacio libre
